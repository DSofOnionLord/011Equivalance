1) equivalence relation을 갖는 입력 데이터를 읽어 equivalence class를 출력하시오. weighted union을 사용하시오.


%cat input.txt
n					// total # of equivalence pairs, where n = 5
0 = 4, 3 = 1, 6 = 10, 3 = 9, 7 = 4	// equivalence pairs



%cat output.txt
3				// total # of equivalence classes, where n =3
{0, 4, 7}, {1, 3, 9}, {6, 10}	// print equivalence classes
